# Data Room

## Run tasks

To install dependencies, use:

```sh
npm install
```

To run the dev server for your app, use:

```sh
npx nx serve app
```

To create a production bundle:

```sh
npx nx build app
```

To run the dev server for your api, use:

```sh
npx nx serve api
```

To create a production bundle:

```sh
npx nx build api
```
